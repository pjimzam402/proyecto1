¡XAMPP PARA WINDOWS!

Para descargar e instalar XAMPP en windows debemos seguir una serie de pasos muy sencillos:

  1. Buscamos "XAMPP windows" en nuestro navegador
  2. Entramos a la página pinchando aquí
  3. Buscamos en la página principal el botón descargar XAMPP para windows
  4. Procedemos a descargarlo
He creado un pequeño tutorial con imágenes para que se entienda mejor.

Entrando en el enlace que he proporcionado arriba entráis directamente a la página principal para descargar XAMPP

Ahora solo pulsad en XAMPP para windows y empezará la descarga

Abrid el instalador cuando haya terminado la descarga y que empiece la instalación de XAMPP

Ahora empezada la instalación nos enseñan los componentes que podemos quitar ya que nos interesan, en mi caso he dejado todos

Ya solo queda seleccionar en carpeta instalaremos XAMPP y ¡a ejecutarlo!

Ya tenemos nuestro XAMPP instalado

¡XAMPP PARA GNU/LINUX!

El primer paso que debemos de realizar para poder tener XAMPP instalado en nuestro sistema es dirigirnos a su web oficial y descargar el instalador que nos ofrecen para Linux. Después abriremos una terminal y le daremos permisos de ejecución al archivo que descargamos, lo hacemos de la siguiente manera.

Para instalar XAMPP abrimos el instalador despues de haberle dado los permisos de ejecución con el comando "chmod 777"

Procedemos a darle a siguiente si queremos la instalación por defecto hasta que termine la instalación

¡Ya podemos ejecutar XAMPP en Linux!

Ejecutar código PHP embebido

Aquí voy a proporcionar unos consejos para ejecutar código php embebido y no cometer errores:

  • Utiliza sintaxis alternativa de PHP- Info
  • Siempre utiliza la sintaxis multi línea para estructuras de control
  • El hecho de que uses etiquetas de HTML y PHP en un mismo archivo, no quiere decir que debas escribir HTML dentro de las sentencias de PHP
  • Mirar el tutorial de W3schools
